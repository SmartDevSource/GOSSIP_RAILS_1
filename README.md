## Installation
```
  gem install bundler
  bundle install
```
Vous devez posséder postgresql et faire appel à ces commandes dans le terminal :
```
  rails db:create
  rails db:migrate
  rails db:seed
```
    
Connectez-vous en local sur le port 3000 avec le lien suivant :
```
  http://localhost:3000/
```
